# Cub3D
Cub3D is a simple 3D game engine based on raycasting technique, inspired by classic games like Wolfenstein 3D. It allows players to navigate through a 2D map rendered in a 3D perspective.
# Features
 - Raycasting
 - Texture mapping
 - Mini-map
 - Player movement and rotation
 - Configurable via map files
# How to run
 - Clone the project
 - navigate to the project folder
 - run the command
   ```bash
   make
   ```
   or, if you want to build the bonus part as well
   ```bash
   make bonus
   ```
 - run the command
   ```bash
   ./cub3D [path_to_the_map_file]
   ```
   There are some example map files in the maps/ folder.
# Controls
 - W: Move forward
 - S: Move backward
 - A: Strafe left
 - D: Strafe right
 - Left Arrow: Rotate left
 - Right Arrow: Rotate right
 - ESC: Exit the game
